## ACCESSIBILITY COMPLIANCE

### WCAG 2.1 AA Compliance
Cart Battery Nation is committed to WCAG 2.1 Level AA accessibility standards:

#### Perceivable
✓ Text alternatives for non-text content
✓ Captions and alternatives for multimedia
✓ Content adaptable to different presentations
✓ Content distinguishable (color contrast 5.2:1 light, 4.8:1 dark)

#### Operable
✓ Keyboard accessible functionality
✓ Sufficient time for content interaction
✓ No content that causes seizures
✓ Navigable with multiple methods
✓ Input modalities beyond keyboard

#### Understandable
✓ Readable and understandable text
✓ Predictable web page behavior
✓ Input assistance and error prevention
✓ Clear form labels and instructions

#### Robust
✓ Compatible with assistive technologies
✓ Valid HTML and ARIA implementation
✓ Future-compatible design patterns

### Accessibility Testing
Regular accessibility audits using:
- Automated testing (axe, WAVE, Lighthouse)
- Manual keyboard navigation testing
- Screen reader testing (NVDA, JAWS, VoiceOver)
- Color contrast verification
- Mobile accessibility testing

### Accessibility Statement
Cart Battery Nation is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply relevant accessibility standards.
